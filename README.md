# Robot NRF

## MicroPython robotics platform with RP2350 Pico2 and ESP32-S3


## Notes

1. Will use analog joysticks for the time being with possibility for using game controller with USB in the future.

2. Possibility to use ASYNC-IO also in the future but depends on how complicated the robot will be and processing requirements.

3. Pin definition files allow the code to use more than one dev board.
